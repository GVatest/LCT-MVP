## Start

**_Clone_**

```bash
git clone https://github.com/GVatest/LCT-MVP
cd hedge-fund
```

### Backend

**_Install_**

```bash
pip install -r requirements.txt
```

**_Start_**

Create database

```bash
python manage.py migrate
```

Start server

```bash
python manage.py runserver
```

### Frontend

**_Install_**

```bash
npm install
# or
yarn install
```

**_Start_**

```bash
npm run dev
# or
yarn dev
```
